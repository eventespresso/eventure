Attract more attendees to your events by posting your events on the Eventure.io event directory for free.

== Description ==

ALPHA VERSION! Currently this plugin just displays a random message about Eventure.io in the WordPress admin.

[Eventure.io](https://eventure.io) is a service that helps you attract a bigger audience to your event. You and many other event organizers post your events to Eventure.io. Eventure.io aggregates these events into an event directory for attendees to search for events by category, type, keyword, price and location.

== Installation ==
